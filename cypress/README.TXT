# Proyecto de Pruebas Automatizadas

Este proyecto contiene pruebas automatizadas para el servicio JSONPlaceholder.

## Requisitos Previos
- Node.js instalado
- Cypress instalado (`npm install`)

## Ejecutar las Pruebas
1. Clona el repositorio: `git clone <URL_DEL_REPOSITORIO>`
2. Ingresa al directorio: `cd <DIRECTORIO_DEL_PROYECTO>`
3. Instala las dependencias: `npm install`
4. Ejecuta las pruebas: `npm test`

## Estructura del Proyecto
- **cypress/integration/api**: Contiene pruebas para el API.
- **cypress/integration/ui**: Contiene pruebas para la interfaz de usuario.
